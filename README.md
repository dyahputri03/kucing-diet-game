# 🐱 Kucing Diet Game

**Kucing Diet Game** adalah game sederhana berbasis **HTML5 Canvas dan JavaScript**.  
Tujuan game: membantu si kucing 😺 menjaga diet dengan melompati tikus 🐀 dan mengumpulkan bonus 🧶.

---

## ✨ Fitur
- Mode terang 🌞 dan gelap 🌙  
- Kontrol lompat dengan **Spasi** di PC  
- Kontrol lompat dengan **tap layar** di HP (Android/iPhone)  
- Sistem skor dan reward message saat mencapai skor tertentu  
- Tampilan responsif, bisa dimainkan di browser desktop maupun mobile  

---

## 🎮 Cara Main
1. Buka link GitHub Pages https://dyahputri03.github.io/kucing-diet-game/
2. Tekan **Spasi** (PC) atau **tap layar** (HP) untuk lompat  
3. Hindari tikus 🐀, kumpulkan skor, dan raih reward 🎉  

---

## 📱 Catatan
Game ini sudah mendukung kontrol layar sentuh, sehingga bisa dimainkan di **Android** maupun **iPhone**.
